# 番茄钟 (Tomato Timer) — Anki 插件

> v1.5.0 · 深色卡片 UI 重构版

在 Anki 复习时帮助你保持专注的极简番茄钟,右侧 dock 显示倒计时,左侧 dock 显示当前卡片的复习次数与本次正确率。

---

## 主要功能

- 🍅 25 + 5 默认节奏,可自定义工作 / 休息时长
- 🎯 复习时自动或手动启动番茄钟
- 📊 左侧复习统计:复习次数、已掌握判定、本次正确率
- 📈 每日番茄数统计,自动持久化到本地数据库
- 🎨 深色卡片化 UI,模式 chip、彩色进度条、64px 大字计时器
- ⚙️ 工具菜单可一键显示/隐藏番茄钟与卡片统计窗口
- 🔔 完成时可选提示音

## 1.5.0 重要更新

### UI 重构(深色卡片设计)
- 番茄钟 widget:64px 大字计时器,带 letter-spacing,主按钮高 48px、圆角 10px
- 模式 chip:🧠 专注时间 / ☕ 休息时间,颜色随模式自动切换
- 顶部状态点:运行 / 暂停 / 就绪三态
- 进度条:8px 圆角 + 模式渐变(红→暗红 / 绿→暗绿)
- stat chip 行:今日番茄数 + 下次休息时长
- 卡片复习统计 widget:状态彩色左边框
  - 🆕 橙 = 新卡片
  - 📖 红 = 复习中
  - 🌿 绿 = 已掌握

### Anki 25 兼容性
- `reviewer_did_answer_card` 改用 `(card, ease, state)` 三参签名
- 兼容修复:解决同进程内其他插件(如 momentum_anki)旧签名抛 `TypeError` 导致本插件统计不更新的问题
- 新增 `on_card_changed_outside_reviewer`:浏览器 / 编辑器修改卡片后自动失效缓存,下次显示重新拉数据
- hook 注册改为 `getattr` 兜底,避免 Anki 26 再改名时崩溃

## 系统要求

- Anki **25.02** 或更高
- Windows / macOS / Linux 全平台支持

## 安装方法

1. 下载 `tomato_timer-1.5.0.ankiaddon`
2. 双击文件 → Anki 自动安装并提示重启
3. 工具菜单出现 **🍅 显示番茄钟** 子菜单

## 使用方法

1. 工具 → 🍅 显示番茄钟 → ☑ 显示计时器(开启右侧 dock)
2. 工具 → 🍅 显示番茄钟 → ☑ 显示卡片统计(开启左侧 dock)
3. 进入牌组开始复习,番茄钟与卡片统计自动出现
4. 工作/休息时长、提示音等可在 `config.json` 中调整
